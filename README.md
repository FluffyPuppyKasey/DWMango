# DWMango
A port/recreation of BreadOnPenguins' DWM configuration on MangoWC using Waybar, with my own personal wants added.

# Note that in waybar/style.css you MUST change the path for the import. Change my username to yours or it won't work.

# Requirements
`waybar`
`mangowc`
`mako`
`foot`
`fuzzel`
`swayosd`
`swww`
`python-pywal`

# Optional (change the config!)
`vesktop`
`telegram-desktop`
`qutebrowser`
`neovim`
